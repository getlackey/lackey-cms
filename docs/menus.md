## Submenus

Mostly resolved by `Drill in` (B)

```
|-----------------------|       ====================
| Properties          > | (B)-> | <     PROPERTIES |
|-----------------------|       ====================
                                | ...              |
```

## Selects

### Single choice

Choice (C) with Radio (I) inside, i.e. view as

```
|-----------------------|       ====================
| View as <Developer> > | (C)-> | <        VIEW AS |
|-----------------------|       ====================
                                | ( ) Guest        | (I)
                                |------------------|
                                | (x) Developer    | (I)
```

### Multiple choice

Choice (C) with Seleect (J) inside, i.e. DOW

```
|-----------------------|       =====================
| Repeat DOW <M,W,F>  > | (C)-> | <             DOW |
|-----------------------|       =====================
                                | [x] Monday        | (J)
                                |-------------------|
                                | [ ] Tuesday       | (J)
```

### Pickers

Choice (C) with Link (A) or complex component. It could have search/filter and add button

```
=================
| <        TAGS |
=================
| ...           |
|---------------|       ========================
| Add tag     > |  -->  | <        CHOOOSE TAG |
|----------------       ========================
                        | ___________ [SEARCH] | (K)
                        |----------------------|
                        | Britney              | (A)
                        |----------------------|
                        | Rihanna              | (A)
                        |----------------------|
                        | ...                  |
                        |----------------------|
                        |   [CREATE NEW TAG]   | (F)
                        |----------------------|
```
